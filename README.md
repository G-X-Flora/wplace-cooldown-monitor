# wplace-cooldown-monitor
python and skirk marble v0.91 are needed to run this . ( --remote-debugging-port=端口 --remote-allow-origins=* (每个快捷方式端口独立，注意--前有空格))is needed . automatically watch over the pixel ink cooldown count , the remaining time for cooldown to fully recover  , and you can record the color that you are drawing . 

for example:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --remote-allow-origins=*
